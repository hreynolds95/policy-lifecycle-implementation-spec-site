Detection Flow

  1. DMS-09.01 Monitor GitHub organization membership or IdP user lifecycle export.
  2. DMS-09.02 Match removed users against `document_owner` fields in the registry.
  3. DMS-09.03 Open Policy Team issue listing affected documents and next review dates.
  4. DMS-09.04 Post Slack alert and track replacement-owner onboarding.
