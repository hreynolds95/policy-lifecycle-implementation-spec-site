12 Legacy Archive & Migration

Preserve audit continuity while limiting historical migration scope.

WorkplanPreviousNext

Migration Principles

  • LogicGate remains system of record until migration readiness is proven.
  • Pre-GitHub PDFs and Google Doc redlines remain in Google Drive.
  • `LEGACY-ARCHIVE.md` links historical records; no bulk content migration is required.
  • GitHub commits and diffs become the redline equivalent for Markdown after cutover.
