07 GitHub Pages Portal

Specify the staff-facing portal for read access, request initiation, metadata, and conflict checks.

WorkplanPreviousNext

Feature Set

  • Browsable library of published governed documents with metadata detail pages.
  • Search by title, domain, owner, tier, status, effective date, and next review date.
  • Role-gated change request initiation for Document Owners and Submitters.
  • Self-service new document request form for any employee, routed to Policy Team validity assessment.
  • Conflict check search and attestation before creation request submission.

Exit Criteria
