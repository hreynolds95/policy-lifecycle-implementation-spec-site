Exit Criteria

  • SSO groups mapped to GitHub Teams.
  • CODEOWNERS rules tested for every governed document type.
  • Portal initiation buttons hidden or disabled for unauthorized users.
  • Access gaps documented where GitHub lacks path-level read restrictions.
